# Discord Fonts

Fonts used in Discord:

- `ABC Ginto Nord`
- `gg sans`
- `gg mono`
- `Noto Sans`
- `Source Code Pro`
- `Corinthia`
- `Fraunces`

## How to use

Import stylesheet

```css
@import url(https://itspi3141.github.io/discord-fonts/font.css);
```

Usage example

```css
.example-class {
    font-family: 'ABC Ginto Nord';
}
```
